# Linear Regression

This folder contains the Jupyter notebook **`Linear Regression.ipynb`**, , which demonstrate how to perform simple linear regression using a dataset with one independent variable and one dependent variable.

## Dataset

The dataset used in the notebook typically includes:

- **Feature** – the independent variable (input).
- **Target** – the dependent variable (output) we are predicting.

The notebook walks through loading the dataset, exploring the relationship between the feature and target, and preparing the data for modeling.

## Tasks Performed

1. **Data Loading and Exploration** – Read the dataset into a pandas DataFrame, inspect its structure, and visualize the relationship between feature and target.
2. **Data Preprocessing** – Handle any missing values, split the data into training and testing sets, and (optionally) scale the data.
3. **Model Building** – Fit a simple linear regression model (from scikit-learn or equivalent) to predict the target from the feature.
4. **Model Evaluation** – Assess model performance using metrics like R-squared and Mean Squared Error (MSE), and visualize predicted vs actual values.
5. **Interpretation** – Examine the regression line and coefficients to understand how changes in the feature affect the target.

## Purpose

This notebook(s) are designed as educational resources for understanding linear regression concepts and how to apply them in Python. They are ideal for learners who want a hands-on introduction to regression modeling.
